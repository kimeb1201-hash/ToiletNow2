🚽 Toilet Now

Toilet Now는 이화여자대학교 교내 학생들을 위한 화장실 위치 탐색 서비스입니다.
현재 위치를 기반으로 교내에서 가장 가까운 화장실을 빠르게 찾고, 청결도·혼잡도·이용 가능 여부까지 한눈에 확인할 수 있도록 도와줍니다.

⸻

📌 서비스 소개

Toilet Now는 수업 쉬는 시간, 이동 중, 급하게 화장실이 필요한 순간에
학생들이 교내 화장실 위치와 상태 정보를 쉽고 빠르게 확인할 수 있도록 설계된 이화여대 전용 앱/웹 서비스입니다.

건물별 화장실 위치, 거리, 혼잡도, 청결 상태 등의 정보를 제공하여
학생들의 이동 시간과 불편함을 줄이는 것을 목표로 합니다.
